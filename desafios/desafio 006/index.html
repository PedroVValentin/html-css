Tags em HTML

Em HTML5 podemos inserir o elementos da lista a seguir. Passe o mouse sobre os termos para descobrir qual tag deverá utilizar.

  • Parágrafo
  • Título
  • Imagens
  • Abreviações
  • Código fonte
  • Lista Numerada (ou lista ordenada)
  • Lista com demarcadores (ou lista não ordenada)
  • Linha horizontal
  • Negrito
  • Links
  • Quebra de linha

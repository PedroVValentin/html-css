Código fonte

Para pode usar como código fonte deve-se usar a tag code document.getElementById('teste')

Sem usar a tag "pre" fica assim: num = int(input('Digite um número')) if num % 2 == 0: print(f'O número {num} é PAR') else: print(f'O número {num} é ÍMPAR') print('Fim do programa')

Usando a tag "pre" fica assim: 

        
    num = int(input('Digite um número'))
    if num % 2 == 0:
        print(f'O número {num} é PAR')
    else:
        print(f'O número {num} é ÍMPAR')
    print('Fim do programa')

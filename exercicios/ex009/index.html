Trabalhando com listas

Listas Ordenadas

Listas ordenadas são listas onde a ordem dos itens importam. Para criar uma lista ordenada se usa a tag "ol" e dentro se usa a tag "li" para adicionar cada item:

  1. acordar
  2. ir no banheiro
  3. escovar
  4. estudar programação
  5. lavar a louça
  6. ir buscar a marmita
  7. almoçar
  8. dar aula

Podemos mudar o estilo e de qual numero começa a lista usando type e start dentro da tag ol.:

  1. acordar
  2. ir no banheiro
  3. escovar
  4. estudar programação
  5. lavar a louça
  6. ir buscar a marmita
  7. almoçar
  8. dar aula

Listas não ordenadas

São listas em que a ordem não faz diferença na lista:

  • pão
  • queijo
  • presunto
  • ovo

Lista de definições

Para fazer um lista de definição se usa a tag "dt" para o termo e a tag "dd" para definição:

MOBA
jogo onde dois times se enfrentão cada um jogando com um personagem.
FPS
jogo de tiro em primeira pessoa.

Formatação de textos

Principais formatações

Negrito ou Destaque

Nesse exemplo a palavra TESTE está em negrito mas usando a tag <B> (não semântica.

Ja nesse TESTE estou usando a tag <Strong> (semântica).

Itálico

Para colocar em itálico apenas forma sem semântica basta colocar a tag <i> e fica assim: TESTE

A forma com semântica do itálico é a tag <em> e fica assim TESTE

Texto marcado

Para colocar um marcador de texto basta colocar a tag <mark> e fica assim; TESTE

Uma forma de colocar uma cor diferente é colocando um tag de style dentro da tag mark assim; TESTE

Texto grande e pequeno

Para fazer um texto grande antigame usava-se a tag <big> mas hoje em dia se usa apenas o CSS para se fazer isso, pois se trata apenas de forma; TESTE comparandoTESTE

Para colocar o texto pequeno ainda se usa a tag <small> e fica assim: TESTE TESTE

Texto deletado

Para fazer um texto deletado deve-se usar a tag <del> TESTE

Texto inserido

Para fazer um texto inserido deve-se usar a tag <ins> TESTE

Texto sobrescrito

Para colocar coisas do tipo x elevado a 20, usamos a tag <sup>: x20+2

Texto subescrito

E para colocar texto mnenor em baixo como em H2O usamos a tag <sub>: H2O
